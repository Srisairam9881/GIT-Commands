Git Commands

Here are some commonly used Git commands:

  • git init - Initialize a new Git repository
  • git add <file> - Add file(s) to staging area
  • git commit -m "message" - Commit changes to the repository
  • git status - Check the status of the repository
  • git diff - shows what made changes are done in code
  • git push - Push changes to a remote repository
  • git pull - Pull changes from a remote repository
  • git branch - List, create, or delete branches
  • git merge <branch> - Merge changes from another branch
  • git clone <repository-url> - Clone a repository from a URL

Git Matters

Git is a distributed version control system designed to handle everything from small to very large projects with speed and efficiency. It provides:

  • Version Control
  • Branching and Merging
  • Collaboration
  • Staging Area
  • Local Repository
  • Remote Repository
